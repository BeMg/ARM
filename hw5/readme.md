# ARM hw5

## 題目要求

將一個2*3的矩陣乘上一個3*2的矩陣，最後加上一個2*2的矩陣

## 矩陣初值

A
```
+---+---+---+
| 1 | 2 | 3 |
+---+---+---+
| 3 | 2 | 1 |
+---+---+---+
```

B
```
+---+---+
| 2 | 3 |
+---+---+
| 1 | 2 |
+---+---+
| 5 | 6 |
+---+---+
```
C
```
+---+---+
| 1 | 2 |
+---+---+
| 3 | 4 |
+---+---+
```

## 程式進行過程

按照矩陣的乘法邏輯，拆解成加減乘法後計算，再依照題目要求，存入正確的位置。
